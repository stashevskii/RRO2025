Данный робот создан для участия в категории “Российская Робототехническая Олимпиада 2025. Базовая Средняя”. Он был разработан на базе конструктора Lego Mindstorms Education EV3 45544. В конструкции были использованы:

  • Один Lego Микрокомпьютер EV3 (45500) (Процессор ARM 9 с тактовой частотой 300 МГц. Операционная система на базе Linux. Встроенная память, включающая 16 МБ flash-памяти и 64 МБ оперативной памяти)
  • Два средних сервомотора Lego EV3 (45503) для движения робота
  • Два средних сервомотора Lego EV3 (45503) для захвата больших и маленьких контейнеров
  • Черные колеса Lego (32019)
  • Два датчика цвета Lego (45506) для движения по линии
  • Один датчик цвета Lego (45506) для распознавания цвета маркеров
  • Один ультра-звуковой датчик Lego (45504) для распознавания высоты контейнеров
